# Branding Guide — Immersive BOP_Harvest

## Style direction

Original botanical-industrial identity:

- leaf = Biomes O' Plenty vegetation;
- gear = integration / Immersive Engineering;
- straw/wheat lines = Farmer's Delight;
- chain-link icon = compatibility link.

No dependency textures, icons, models, sounds or logos are copied.

## Included assets

### Vector logo

```text
assets/branding/transparent/immersive_bop_harvest_logo_vector.svg
```

Use for:
- documentation;
- source repository;
- project cards.

### Vector banner

```text
assets/branding/cover/immersive_bop_harvest_banner_vector.svg
```

Use for:
- Notion cover/banner;
- repository and release page headers;
- project status pages.

### Raster assets

The source pack included PNG icons, banners and backgrounds. They are not part
of this repo payload per the current project scope.

## Typography

No font file is included or redistributed.

Recommended public/system fonts:
- title: Montserrat, Inter, Segoe UI, Arial, sans-serif;
- body: Inter, Segoe UI, Arial, sans-serif;
- code: JetBrains Mono, Consolas, monospace.

- Do not combine these assets with copied BOP, Farmer's Delight or Immersive Engineering logos.
