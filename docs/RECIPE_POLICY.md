# Recipe and Loot Policy

## Core rule

A new conversion must be both:
1. materially credible; and
2. lower-value or equal-value compared with existing progression.

## Allowed design patterns

- flower → the dye BOP already assigns to that flower;
- dried grass/reed → Farmer's Delight straw;
- bush/branch/bramble → stick;
- web/silk/sinew/hair → string;
- flesh block → partial rotten flesh;
- log/wood → stripped form + tree bark;
- log/wood → planks + wood dust in the IE Sawmill;
- barley → common grain tag;
- toadstool → common mushroom tag.

## Forbidden design patterns

- decorative plant → gem, ingot, emerald or rare mob drop;
- End plant → ender pearl or chorus fruit;
- Nether flower → blaze powder;
- luminous plant → glowstone dust;
- arbitrary plant → IE hemp fiber or hemp seed;
- reed → sugar cane;
- barley → wheat;
- watergrass → kelp;
- potted plant → bonus resources;
- special encounter block → farmable commodity.

## Probability ceiling

- low-yield botanical direct drops: 25%–50%;
- web-like string with correct cutting tool: 100%;
- no rare secondary outputs;
- no Fortune scaling.

## Existing BOP parity

Never exceed BOP's own dye count in a Cutting Board recipe. The Cutting Board is an alternative workflow, not an efficiency exploit.
